### commands cheatsheet

```sh
vagrant box add debian/bookworm64
vagrant up
vagrant halt
vagrant destroy
vagrant ssh

vagrant status
vagrant reload

ls -la ~/.vagrant.d/boxes
ls -la ~/VirtualBox\ VMs
ls -la ~/.vagrant.d/insecure_private_key
```
